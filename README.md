##v-charts配置  https://v-charts.js.org/
### :extend 所有的配置都可以用这个值重新设置
```
{
  'xAxis.0.axisLabel.rotate': 45,
   yAxis: {
    splitLine: {
      show: true,
      lineStyle: {
        color: '#303055',
        width: 0.5
      }
    }
  },
  textStyle: {
    color: '#fff'
  }
}
```
